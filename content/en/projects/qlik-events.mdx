# Context & role
Claims monitoring for natural events. Role: Data Analyst / BI Consultant.

## Business objective
Provide a consolidated view of volumes, costs and trends to improve decision-making.

## Data & sources
- Claims feeds
- Contract/coverage reference data

## Architecture
- Qlik data model optimized for drill-down
- Standardized KPIs

## Pipeline / processing
1. SQL extraction
2. Qlik modeling
3. Data quality checks

## Modeling / KPI
- Claims count
- Average cost per event
- Processing time

## Dashboard / results
- Global view with event and period filters
- Comparative analyses

## Code snippets
```sql
SELECT event_type, COUNT(*) AS claims
FROM claims
GROUP BY event_type;
```

## Learnings / improvements
- Decision-oriented dashboard design
- Qlik performance optimization
