# Context & role
Mission for a landside cleaning vendor. Role: Data Analyst / Junior Data Engineer, end-to-end pipeline and reporting setup.

## Business objective
Provide daily and weekly views of cleaning operations to anticipate workload and manage costs.

## Data & sources
- Operational files (planning, volumes, zones)
- Zone/activity reference data
- Dataiku datasets

## Architecture
- **Bronze**: raw ingestion
- **Silver**: joins, quality checks, enrichment
- **Gold**: BI-ready business tables

## Pipeline / processing
1. Profiling + validation (Dataiku)
2. SQL/Python cleaning
3. KPI computation by zone and period
4. Publishing Gold tables to Power BI

## Modeling / KPI
- Volume by zone / shift
- Average cost per operation
- Weekly trend

## Dashboard / results
- Terminal comparisons
- Overload detection by zone
- Quality target tracking

## Code snippets
```sql
SELECT zone, date, SUM(volume) AS volume_total
FROM fact_operations
GROUP BY zone, date;
```

## Learnings / improvements
- Automated quality checks
- Standardized KPI rules
- Faster refresh cycles
