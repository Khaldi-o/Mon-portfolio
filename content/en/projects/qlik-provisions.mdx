# Context & role
Claims provisions and recoveries monitoring. Role: Data Analyst / BI Consultant.

## Business objective
Expose provisions, recoveries and budget variances for insurance teams.

## Data & sources
- Claims finance feeds
- Contract and coverage reference data

## Architecture
- Qlik model for multi-dimensional analysis
- Standardized KPIs with quality checks

## Pipeline / processing
1. SQL extraction
2. Normalization and controls
3. Qlik load and aggregations

## Modeling / KPI
- Provisions by portfolio
- Recoveries
- Monthly variance

## Dashboard / results
- Global view + drill-down
- Variance tracking
- Anomaly alerts

## Code snippets
```sql
SELECT portfolio, SUM(provision) AS provision_total
FROM claims_finance
GROUP BY portfolio;
```

## Learnings / improvements
- Align finance definitions
- Strengthen upstream controls
