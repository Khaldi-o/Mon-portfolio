# Contexte & role
Pilotage des provisions et recours sinistres. Role : Consultant Data Analyst / BI.

## Objectif metier
Mettre en visibilite les provisions, recours et ecarts budgetaires pour aider les equipes assurance.

## Donnees & sources
- Flux sinistres et provisions
- Referential contrats / garanties

## Architecture
- Modele Qlik pour analyses multi-dimensions
- KPI standardises et verifications qualite

## Pipeline / traitement
1. Extraction SQL des flux
2. Normalisation et controles
3. Chargement Qlik et agregations

## Modelisation / KPI
- Provisions par portefeuille
- Recours recuperes
- Variation mensuelle

## Dashboard / resultats
- Vue globale + drill-down
- Suivi des ecarts
- Alertes sur les anomalies

## Extraits code
```sql
SELECT portfolio, SUM(provision) AS provision_total
FROM claims_finance
GROUP BY portfolio;
```

- Fiabiliser les controles amont
