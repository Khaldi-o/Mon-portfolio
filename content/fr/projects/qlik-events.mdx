# Contexte & rôle
Pilotage des sinistres liés aux événements naturels. Rôle : Consultant Data Analyst / BI.

## Objectif métier
Offrir une vision consolidée des volumes, coûts et tendances pour améliorer la prise de décision.

## Données & sources
- Flux sinistres
- Référentiels contrats / garanties

## Architecture
- Modèle de données Qlik optimisé pour le drill-down
- KPI standardisés

## Pipeline / traitement
1. Extraction SQL
2. Modélisation Qlik
3. Contrôles qualité

## Modélisation / KPI
- Nombre de sinistres
- Coût moyen par événement
- Délai de traitement

## Dashboard / résultats
- Vue globale avec filtres par événement et période
- Analyses comparatives

## Extraits code
```sql
SELECT event_type, COUNT(*) AS nb_sinistres
FROM claims
GROUP BY event_type;
```

## Ce que j’ai appris / améliorations
- Structurer des dashboards orientés décision
- Optimiser les performances Qlik
